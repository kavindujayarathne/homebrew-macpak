# Homebrew Tap: macpak

This is the official [Homebrew tap](https://docs.brew.sh/Taps) for [macpak](https://github.com/kavindujayarathne/macpak).

## Installation

```bash
brew install --formula kavindujayarathne/macpak/macpak
```

## Install from HEAD

```bash
brew install --HEAD macpak
```
